# 武将单挑机制设计文档

**日期**: 2025-02-05
**状态**: 设计阶段

---

## 一、核心概念

单挑是两名武将之间的 1v1 格斗，独立于大军战斗之外，但结果会影响战场态势。

### 设计原则

- 参考《三国志 11》的单挑机制，结合《三国群英传》的即时战斗
- 即时自动战斗 + 玩家干预
- 可换人的 3v3 格式，但场上永远是 1v1

---

## 二、触发场景

| 场景 | 触发方式 |
|------|----------|
| 战场主动 | 玩家/AI 在带兵对战时发起单挑邀请 |
| 战场随机 | 一方发动特定战法时，有概率触发单挑 |
| 特定事件 | 攻城战、遭遇战、剧情事件 |
| 比武大会 | 一年一度的专门赛事 |

---

## 三、单挑格式与换人

### 单挑格式

| 格式 | 说明 |
|------|------|
| 1v1 | 最基本的单挑 |
| 3v3 | 可换人，场上永远 1v1 |

### 换人机制

- **随时可换**：不限制换人次数和时机
- **体力保留**：下场武将保留当前体力
- **豆子继承**：新上场的武将继承当前豆子数量
- **已下场武将**：本场单挑不能再上场

### 策略点

换人时机是核心策略：
- 太早换 → 浪费优势武将
- 太晚换 → 被打败来不及换

---

## 四、战斗方式

**即时自动战斗 + 玩家干预**

- 两个武将按照预设的**战斗方针**自动交战
- 玩家/AI 可以实时切换方针
- 玩家/AI 可以手动释放技能（消耗豆子）

### 方针系统

| 方针 | 效果 |
|------|------|
| 重视进攻 | 攻击频率↑，暴击率↑，斗志积累↓ |
| 重视防御 | 闪避率↑，减伤↑，斗志积累↓ |
| 重视斗志 | 攻击频率↓，斗志积累↑ |

---

## 五、胜负判定

```
场上武将体力 <= 0
    ↓
该武将所在队伍直接败北
    ↓
对方队伍获胜
```

### 示例

- A1 vs B1，B1 残血时换 B2
- B2 把 A1 打败（体力归零）
- A 队伍输了（即使 A2、A3 没上过场）

---

## 六、属性影响

| 属性 | 影响单挑 |
|------|----------|
| **勇武** | 基础攻击力、暴击率、暴击伤害 |
| **智力** | 闪避率、技能释放速度、斗志积累效率 |
| **统帅** | 单挑胜利后对己方部队的士气加成幅度 |

### 体力系统

- 每个武将有独立的 **体力值**（基于勇武或单独属性）
- 体力归零 = 败北
- 换人时保留当前体力（不会自动回复）

---

## 七、技能系统

### 豆子获取

- 自动积累斗志，**100 斗志 = 1 豆子**
- 不同方针影响积累速度：
  - 重视斗志：积累最快
  - 重视进攻/防御：积累较慢
- **豆子上限：3 个**

### 基础技能

| 豆子消耗 | 伤害倍率 | 额外效果 |
|----------|----------|----------|
| 1 个 | 基础伤害 | 无 |
| 2 个 | 1.5 倍 | 有概率使对方**负伤** |
| 3 个 | 3 倍 | 更高概率使对方**负伤** |

**负伤效果**：降低武将的战斗力（攻击力、移动速度等），持续一定时间或直到单挑结束

### 技能示例

**2 豆子技能**：
- 横扫千军：1.5 倍伤害，负伤概率中等
- 突刺：1.5 倍伤害，破防（无视部分防御）

**3 豆子必杀技**：
- 温酒斩华雄：3 倍伤害，高负伤概率
- 千人斩：连续攻击，累积伤害约 3 倍

---

## 八、武器加成

武器可以提供单挑时的额外效果：

| 武器效果 | 说明 |
|----------|------|
| 斗志上升 | 斗志积累速度提升 |
| 武力上升 | 单挑时临时增加勇武值 |
| 负伤率上升 | 2/3 豆子技能造成负伤的概率提升 |
| 体力加成 | 单挑时体力上限提升 |
| 体力回复 | 每回合缓慢恢复体力 |

---

## 九、单挑熟练度

武将参与单挑可获得熟练度经验，熟练度等级解锁新技能：

| 熟练度等级 | 解锁内容 |
|------------|----------|
| Lv.1 | 初始技能（1/2/3 豆子基础攻击） |
| Lv.3 | 解锁 2 豆子新技能 |
| Lv.5 | 解锁 3 豆子必杀技 |
| Lv.7 | 解锁被动能力（如低血量时攻击力提升） |
| Lv.MAX | 解锁专属绝技 |

### 专属绝技示例

- 关羽：**青龙偃月** — 巨量伤害，低血量时暴击率翻倍
- 张飞：**狮子吼** — 伤害 + 降低对方攻击力
- 赵云：**七进七出** — 连续 7 次攻击，每次伤害递增

## 十、胜负结果与战场影响

### 单挑结果类型

| 结果 | 说明 |
|------|------|
| **击败** | 对方体力归零，获得胜利 |
| **负伤** | 对方体力未归零但受到负伤状态，单挑结束后持续影响 |
| **斩杀** | 对方体力归零且触发斩杀，武将直接死亡 |
| **俘虏** | 对方体力归零且触发俘虏，武将被俘（受魅力影响） |

### 对战场的影响

单挑结果直接影响双方部队的士气和状态：

| 结果 | 己方部队 | 敌方部队 |
|------|----------|----------|
| **击败** | 士气 +20 | 士气 -20 |
| **斩杀** | 士气 +30 | 士气 -40，混乱 |
| **俘虏** | 士气 +40 | 士气 -50，溃散 |
| **己方被击败** | 士气 -30 | 士气 +30 |
| **己方被斩杀** | 士气 -50，溃散 | 士气 +40 |

**统帅的影响**：统帅高的武将单挑胜利后，对己方部队的士气加成更高

### 逃跑机制

- 单挑中可以选择**认输逃跑**
- 逃跑成功率 = 基础率 + 智力加成 - 对方勇武加成
- 逃跑成功：武将存活但士气大幅下降
- 逃跑失败：受到额外伤害，且士气继续下降

## 十一、设计总结

| 模块 | 内容 |
|------|------|
| 核心概念 | 即时自动战斗 + 玩家干预方针和技能 |
| 队伍格式 | 1v1 或 3v3（可换人，场上永远 1v1） |
| 胜负判定 | 场上武将体力归零 = 该队伍败北 |
| 方针系统 | 重视进攻/防御/斗志，影响战斗节奏 |
| 技能系统 | 1/2/3 豆子消耗，伤害递增 + 负伤概率 |
| 武器加成 | 斗志/武力/负伤率/体力等提升 |
| 熟练度成长 | 解锁新技能和被动能力 |
| 战场影响 | 士气变化、斩杀/俘虏、逃跑 |
